Ciao! In questo corso, l'obiettivo principale dei nostri studenti è creare progetti
interessanti utilizzando Python e le sue librerie! Per quanto riguarda il vostro
compito pratico, provate a utilizzare le API e le funzionalità di database di Flask per
creare un sito web di quiz dinamico e accattivante rivolto a un pubblico di
adolescenti!
Possibili argomenti per un quiz:
· Sviluppo di intelligenza artificiale in Python
· Visione computerizzata
· NLP (Elaborazione del Linguaggio Naturale)
· Applicazione di modelli di intelligenza artificiale alle applicazioni Python

Criteri di valutazione dei risultati:
· Il sito web presenta un quiz su uno degli argomenti suggeriti,
rilevante per il pubblico di riferimento.
· Il sito web presenta un menu di navigazione
· Il sito web presenta un widget per le previsioni del tempo
· Il piè di pagina del sito web contiene informazioni sull'autore
· Il sito web presenta pagine di registrazione e login funzionanti
· Il quiz è composto da diverse domande
· Una classifica viene aggiornata ogni volta che un utente supera il
quiz
· Il sito web è distribuito su PythonAnywhere

Il sito web deve presentare pagine con i seguenti contenuti:
Home page:
· Un campo di immissione per il nome della città
· Una tabella con le previsioni del tempo per 3 giorni (oggi, domani e
dopodomani):
· La pagina web dovrebbe fornire la data, compreso il nome del giorno
(lunedì, martedì, ecc.) La home page dovrebbe anche informare l'utente
della temperatura diurna e di quella notturna.
Pagina di registrazione:
· Campo di login (il sito web deve verificare se il login è unico)
· Il campo della password
· Il campo di conferma della password
· Il campo del nickname (il sito web dovrebbe verificare se il login è unico)
Pagina di login:
· Il campo di login
. Il campo della password

II Quiz:
· La pagina deve visualizzare il punteggio totale del giocatore dal momento
della registrazione (questo valore deve essere collegato all'account nel
database).
· La pagina deve contenere una domanda ...
· ... e 4 opzioni di risposta
(Nota: le domande devono essere poste all'infinito e in ordine casuale).
La classifica dei quiz:
· La classifica deve visualizzare i nomi e i punteggi dei giocatori.
Ogni pagina deve presentare un'intestazione con un menu di navigazione:
· Pagina iniziale

· La finestra di registrazione (visibile se l'utente non ha effettuato il login)
· La finestra di login (visibile se l'utente non è connesso)
· La finestra di logout (visibile quando l'utente è connesso)

· Il Quiz (visibile quando l'utente è connesso)
Ogni pagina deve mostrare un piè di pagina con il nome dello sviluppatore.

Il sito web finito deve essere ospitato sul servizio di hosting gratuito
PythonAnywhere.com.

IMPORTANTE: prima di inviare il progetto, assicuratevi di testarlo completamente.
Verificate che tutti i criteri siano stati soddisfatti, che tutti gli elementi richiesti siano
presenti nelle pagine e che il progetto funzioni senza errori.
Una volta creato il sito web, nel test devono essere allegate le seguenti informazioni
(separare le voci 1 e 2 con una virgola)
1. Un link alla pagina iniziale

2. Un link alla cartella dei file del progetto nel cloud.
